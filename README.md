# TIPatcher
"Patches" TI Nspire CX CAS Student Software 

## Prerequisites
- A computer running Windows
- TI Nspire CX CAS Student Software installed

## How to use
1) Download the latest release (patcher.zip) from Releases
2) Download and install the [TI Nspire CX CAS Student Software](https://education.ti.com/en/software/details/en/36BE84F974E940C78502AA47492887AB/ti-nspirecxcas_pc_full)
3) Extract patcher.zip and run TIPatcher.exe
